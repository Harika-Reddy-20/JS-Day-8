Split:

converts string to an array, split method returns the new array, does not the change the original one.

 let myStr = 'Harika'
                let sp = myStr.split("");
                console.log(myStr); 
                console.log(sp); // ['H','a','r','i','k','a']

Reverse and join:

It reverse the string & It converts array to string.

                    let res = myStr.split("").reverse();
                    console.log(res); 
                    o/p:-['a','k','i','r','a','H']
                    let res1 = myStr.split("").reverse().join("");
                    console.log(res1);
                    o/p:- akiraH

Slice:

It is used to extract data, it will not modify the original.

  let str1 = ["hi", "hello","how", "are", "you"]
                      let res = str1.slice(1,3);
                      console.log(res); 
                      o/p:- (2) ['hello', 'how']

Splice:

It is used for extraction and adding the elements and it will modify the original.

    let str1 = ["hi", "hello","how", "are", "you"]
   
                        let res=str1.splice(1,3);
                        console.log(res); 
                        o/p:- (3) ['hello', 'how', 'are']
